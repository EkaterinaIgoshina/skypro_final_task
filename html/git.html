Git

  • git clone - для клонирования репозитория.
  • git add - для добавления изменённых файлов в индекс.
  • git commit - для сохранения изменений в локальном репозитории.
  • git push - для отправки изменений на удалённый репозиторий.
  • git pull - для получения изменений из удалённого репозитория/
